  <action>
    Update src/api/extract.py extract_entities function per D-12:
    
    1. Import asyncio at top of file.
    
    2. Make extractor.extract() calls async. Wrap synchronous extract calls: temporal_task = asyncio.to_thread(temporal_extractor.extract, request_body.text); clinical_task = asyncio.to_thread(clinical_extractor.extract, request_body.text). Use to_thread because llama-cpp-python is synchronous (not async), but we want parallel execution.
    
    3. Initialize both extractors: temporal_extractor = extractor_registry["temporal"](request.app.state.model); clinical_extractor = extractor_registry["clinical"](request.app.state.model).
    
    4. Run in parallel: temporal_result, clinical_result = await asyncio.gather(temporal_task, clinical_task) per D-12.
    
    5. Merge results: return EntityResponse(temporal_entities=temporal_result["temporal_entities"], clinical_entities=clinical_result["clinical_entities"], errors=[], low_confidence=[]).
    
    6. Update error handling to preserve partial results per D-05: wrap each gather call in try/except, accumulate errors, return EntityResponse with partial entities + errors array.
    
    Update tests/test_extraction.py:
    
    Add **test_extract_all_entity_types:** POST /extract with German text containing dates, diagnoses, medications, LOS. Assert response has len(temporal_entities) >= 2, len(clinical_entities) >= 2. Verify each entity has type, text, confidence, source_span fields.
